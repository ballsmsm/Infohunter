# InfoHunter

**Advanced OSINT & Recon tools for Termux**  
*Status: Beta*

## Features
- IP & Domain OSINT
- Username & Email OSINT
- Discord & Roblox OSINT
- Network & Port Scanning
- Credentials & Hash checks

## Installation
```bash
git clone https://github.com/ballsmsm/InfoHunter.git
cd ~/ispeaks-kali
python3 ispeaks-kali.py
pkg install python
python3 -m ensurepip
